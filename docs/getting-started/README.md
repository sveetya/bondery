# Getting Started

This section covers everything you need to set up Bondery locally for development.

## Prerequisites

- **Node.js** >= 20
- **npm** >= 10 (ships with Node.js 20+)
- **Supabase CLI** -- for running the local database ([install guide](https://supabase.com/docs/guides/cli/getting-started))
- **Docker** -- required by Supabase CLI for local development
- **Git**

### Optional

- **GitHub OAuth app** -- for login via GitHub ([docs](https://docs.github.com/en/apps/oauth-apps/building-oauth-apps/creating-an-oauth-app))
- **LinkedIn OAuth app** -- for login via LinkedIn OIDC
