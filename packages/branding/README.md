# @bondee/branding

Bondee branding assets and icons package.

## Usage

### Import the SVG icon

```typescript
import iconPath from '@bondee/branding/icon';
```

### In Node.js scripts

```javascript
const path = require('path');
const iconPath = path.join(require.resolve('@bondee/branding'), '../icon.svg');
```

## Assets

- `icon.svg` - Main Bondee icon (128x128)
